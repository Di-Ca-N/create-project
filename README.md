# Create Project

Script to create python projects already configured with virtualenv and some packages (according custom templates)

## Registered templates
|Template          |Installed Packages   |
|------------------|---------------------|
|django            |django               |
|django-postgres   |django, psycopg2     |
|django-mysql      |django, mysqlclient  |
|scientific        |pandas, matplotlib   |
|basic-ia          |scikit-learn, pandas |
